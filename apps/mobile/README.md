# NearMe Mobile

App Expo do MVP do NearMe.

## Setup

1. Copie `.env.example` para `.env`
2. Confirme `EXPO_PUBLIC_API_URL=http://localhost:3002`
3. Instale dependencias na raiz do monorepo

## Rodar

- `npm run dev:mobile`
- `npm --prefix apps/mobile run android`
- `npm --prefix apps/mobile run ios`

## Fluxos conectados

- login e cadastro via backend local
- edicao de perfil
- toggle de visibilidade
- pessoas proximas
- pedidos de conexao
- aceite e visualizacao de conexoes
